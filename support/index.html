Help with NoteHere

Enable NoteHere

  1. Open the NoteHere app.
  2. Choose Open Safari Settings.
  3. Enable NoteHere under Extensions. Safari adds NoteHere's button to the toolbar automatically.
  4. Allow access to websites when Safari asks.
  5. On a normal webpage, open NoteHere's toolbar menu and choose New Note.

If you previously removed the button, choose View > Customize Toolbar in Safari and drag NoteHere back to the toolbar. Safari controls toolbar customization, so NoteHere cannot override a button the user removed.

Protected Pages

Safari does not permit extensions to run on Safari settings, start pages, or some protected internal pages.

Privacy And Deletion

Notes stay locally on your Mac. Use Delete Notes on This Page in Safari or Delete All in the manager. Read the Privacy Policy.

Get Help

Report a problem at GitHub Support Issues. Include your macOS version, Safari version, the type of webpage, and steps to reproduce. Do not include private note text or confidential URLs.
