Privacy Policy

NoteHere for Safari and Chrome

Effective July 22, 2026.

Summary

The NoteHere extensions do not collect, track, sell, transmit, or share personal data. They have no accounts, analytics, advertising, payment system, cloud synchronization, or automatic external network service.

Data Stored On Your Device

NoteHere stores note text and formatting, checklist state, note color and position, creation and modification dates, settings, and the exact webpage URL associated with each note. The Safari version stores this data in Safari extension storage and NoteHere's local shared container on your Mac. The Chrome version stores it locally in the browser profile using Chrome extension storage and IndexedDB.

Website Access

Safari and Chrome website access lets NoteHere display its isolated note interface and identify the exact URL where you created a note. NoteHere does not inspect or store webpage text, form contents, passwords, cookies, or general browsing history. Your browser controls website permissions and protected pages.

Exports And Deletion

Note data is written outside local extension storage only when you explicitly export a backup or readable text file. You can delete individual notes or every note on the current exact page. Chrome users can also search and delete notes from NoteHere Settings.

This Support Website

Opening this privacy policy, the support page, or the GitHub issue tracker is a normal browser navigation that you choose. These public pages are hosted by GitHub Pages and are subject to GitHub's privacy statement. The NoteHere extensions do not send note contents or exact note URLs to this website.

Contact

Questions can be submitted through the NoteHere support page.
